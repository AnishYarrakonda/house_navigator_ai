Waypoint Design System

The calm, map-first system for finding somewhere safe tonight.

A premium dark foundation built for pressure. The map is home on every side; the interface stays quiet around it — hairline borders, glass surfaces, a single restrained accent, and routes that glow only when a path home is growing.

Trauma-informed WCAG AA · 44px targets Icon-first · EN / ES
01 — Foundations

Color & light

One cool, near-black world. Surfaces step up by light, not by hue. The indigo accent is rationed — used for the primary action and a person's living route, almost nowhere else. Capacity is the only color that carries meaning, and it's always paired with a label and shape.

02 — Foundations

Typography

An editorial pairing — chosen so the product reads as human and unmistakably ours, not another Inter or Geist interface. Headlines are set in a serif, DM Serif Display; everything you read and tap is Hanken Grotesk; counts, distances and codes use Space Mono. Want a different voice?
